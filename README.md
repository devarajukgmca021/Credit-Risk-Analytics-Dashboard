# Credit Risk Analytics Dashboard

## Project Overview
This project analyzes a banking credit risk dataset (12,000+ records, 22 features) to evaluate loan portfolio health, default risk patterns, and financial risk drivers.

Built using:
- Power BI (Interactive Dashboard)
- Python (Advanced Statistical Analysis)
- DAX (Risk Metrics Calculation)

---

## Dashboard Structure

### Page 1: Executive Credit Risk Overview
- Total Loans Issued
- Portfolio Value
- Default Rate
- Average Credit Score
- Loan Status Distribution

### Page 2: Regional Risk & Portfolio Dynamics
- Regional Loan Growth
- Default Rate by Region
- Risk Heatmap
- Income vs Loan Exposure Analysis

### Page 3: Advanced Credit Risk Intelligence
- Correlation Matrix
- Debt-to-Income Impact Analysis
- Risk Category Distribution

### Page 4: Advanced Statistical & Relationship Analysis (Python)
- Pair Plot
- Violin Plot
- Correlation Heatmap
- Network Graph

---

## Key Insights
- Strong correlation between Debt-to-Income Ratio and Default Risk
- Region-based risk concentration observed
- Credit score distribution significantly differs across loan status
